## Features

- End-to-end machine learning workflow for churn prediction

- Training and evaluation of classification models
- Identification of key factors influencing customer churn

## Dataset Overview

The dataset contains telecom customer information such as:

- Customer demographics and tenure
- Service subscription details
- Billing and account information
- Churn label indicating customer attrition
